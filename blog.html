Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is used to represent web documents. A tree of objects, reflects the hierarchical structure of a document(e. g., an XML or HTML file). Every node is an element, attribute, or piece of text in the document. The DOM is a collection of methods and object properties that are used to access the content, structure and style of a document using languages like JavaScript.

How do you select an element from the DOM?

We can select elements from the DOM using various methods in JavaScript. Here are some common ones:
const element = document.getElementById('Id');
const elements = document.getElementsByClassName('Class');
const elements = document.getElementsByTagName('div');
const element = document.querySelector('.myClass');
const elements = document.querySelectorAll('div.myClass');

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a common practice where we add all the event listeners to parent element and handle on child elements. So if an event happens on a child element, eventually it bubbles up to the parent where any one of these event listeners can pick that up. Benefits of event delegation: This improves performance by decreasing the number of event listeners in the DOM, especially when there are many elements. Dynamic Elements: It Works Automatically for Dynamic child elements as listener will always add to current available element and listening dynamic control which are going get added in future thus works without use of hooks, We can removeEventListener on unmount if user want. Maintenance: Easy to maintain the code as well understand and update event handling.

How do you manipulate an element's attributes and styles using the DOM?

We can manipulate an element's attributes and styles using JavaScript as follows:
Set an attribute:
const element = document.getElementById('myElement'); br element.setAttribute('data-custom', 'value');
Get an attribute:
const value = element.getAttribute('data-custom');
Remove an attribute:
element.removeAttribute('data-custom');
Set a style:
element.style.backgroundColor = 'blue';
element.style.fontSize = '16px';
